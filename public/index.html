Voting Online and Tools for Election

VOTE is a web app that makes casting a vote easy and simple. The user can login/register into the application which uses openCV to make the authentication safe. The user can chat in real-time with other people to discuss politics and certain issues. They can also check nearby polling locations along with current elected representatives.
